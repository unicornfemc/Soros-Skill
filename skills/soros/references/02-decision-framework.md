# 索罗斯决策框架与 BISES 测试

## 五步决策框架详解

### 第一步：识别主导偏见（Identify the Prevailing Bias）

**核心问题：** 市场现在普遍相信什么？这个共识是否在根本上是错误的？

**操作步骤：**
1. 列出当前市场三大主导叙事
2. 对每条叙事问：这个叙事成立的前提条件是什么？这些条件可持续吗？
3. 识别最容易被证伪的那条叙事 → 这是潜在的高赔率交易机会

**典型错误偏见案例：**
- 1980s日本：「地价只会涨不会跌，因为日本情况特殊」
- 1990s初欧洲：「欧洲汇率机制（ERM）是不可逆的」
- 1999互联网：「新经济没有周期」
- 2006美国：「房价只会上涨，次贷风险可控」
- 2010s欧洲：「欧洲央行不会打破「不惜一切代价」（OMT）」

---

### 第二步：定位反射循环（Locate the Reflexive Feedback Loop）

**两种反馈方向：**

**A. 认知功能主导的反馈（Bias → Prices → Fundamentals）**
- 预期变好 → 资产价格上涨 → 财富效应 → 消费/投资增加 → 预期进一步变好
- 适用场景：宽松货币政策初期、趋势启动阶段

**B. 参与功能主导的反馈（Fundamentals → Prices → More Fundamentals）**
- 企业盈利改善 → 股价上涨 → 并购成本降低 → 更多整合 → 盈利进一步改善
- 适用场景：趋势中后段、自我实现的预言阶段

**识别方法：** 问自己——"价格上涨是因为基本面真的变了，还是因为大家预期它会涨？预期它会涨这件事本身，有没有进一步改变了基本面？"

---

### 第三步：BISES 测试

**BISES = 国际收支（Balance of payments）、通胀（Inflation）、经济状态（Economy）、退出机制（Exit）、可持续性（Sustainability）**

| 维度 | 高反射性信号 | 低反射性信号 |
|---|---|---|
| **国际收支** | 经常账户持续恶化、外部融资依赖度高、资本流入主要由利差驱动 | 顺差稳定、外汇储备充足、外资配置多元化 |
| **通胀预期** | 工资-价格螺旋启动、长期通胀预期脱锚、央行信誉受损 | 通胀预期稳定锚定、央行有清晰通胀目标框架 |
| **经济状态** | 信贷增速远超GDP增速、资产价格与实体背离、产能利用率极端 | 信贷/GDP比率健康、资产价格与收益匹配 |
| **退出机制** | 央行弹药耗尽、财政空间耗尽、政策空间受限 | 利率工具充裕、财政有余地、政策响应空间大 |
| **可持续性** | 趋势延伸已大幅超出历史极限、杠杆率持续攀升 | 估值接近历史中枢、杠杆率可控 |

**评分：** 5项中≥3项显示高反射性 → 高度关注趋势交易机会

---

### 第四步：超级周期定位

**三阶段模型：**

| 阶段 | 特征 | 索罗斯策略 |
|---|---|---|
| **早期（Early）** | 偏见明显但不被接受；市场仍在争论 | 小仓位试探，确认偏见后加仓 |
| **中期（Amplification）** | 偏见成为共识；反馈循环自我强化 | 持有核心仓位，趋势跟随者入场放大趋势 |
| **晚期（Late Cycle）** | 极端偏离、退出机制受限、原则健全性承压 | 逐步减仓、设置紧密止损、准备做空 |

**判断方法：** 
- 叙事是否已经从"有争议"变成"显而易见且不可挑战"？
- 反对意见是否已经从"合理的怀疑"变成"被认为愚蠢"？
- 杠杆使用是否已经极端？

---

### 第五步：仓位结构与退出策略

**入场原则：**
- 必须在止损明确的前提下入场
- 止损点 = "如果这件事发生，说明我的论点根本性错误"
- 不是技术止损，是基本面止损

**仓位法则：**

| 信念程度 | 仓位建议 | 止损幅度 |
|---|---|---|
| 试探性（Thesis unconfirmed） | 1-2% | 5-10% |
| 确认性（Thesis confirmed, early） | 5-10% | 10-15% |
| 高度确信（Full validation） | 15-25% | 15-20% |
| 极端确信（Super-cycle conviction） | 25-40% | 20-30% |

**关键规则：当你极度确信时——下重注。错误时要果断止损，不要摊平成本。**

**退出触发类型：**

| 退出类型 | 触发条件 |
|---|---|
| **时间止损** | 超过预定时间（X周/季度）趋势未启动 |
| **事件止损** | 特定事件发生（政策转向、数据恶化） |
| **自我否定** | 出现与原有论点矛盾的关键证据 |
| **趋势终止** | 反射性逆转信号出现（见03文件） |

## 实战工作流

```
1. 宏观事件/市场异动出现
         ↓
2. 问：这是「正常的市场噪音」还是「反射性趋势的可能起点」？
         ↓
3. 如果是潜在机会 → 做 BISES 五维评估
         ↓
4. 如果 BISES ≥ 3项高反射性 → 识别主导偏见
         ↓
5. 定位反馈循环方向 → 超级周期定位
         ↓
6. 确定止损点 → 设定仓位 → 入场
         ↓
7. 每日监控：反射性信号是否持续？Breaking point 是否临近？
```

## 索罗斯 vs 巴菲特决策框架对比

| 维度 | 索罗斯 | 巴菲特 |
|---|---|---|
| 分析起点 | 市场共识/偏见是什么？ | 标的本身价值是多少？ |
| 核心能力 | 识别叙事错误 + 时机 | 商业质量评估 + 耐心 |
| 错误时的反应 | 立即止损 | 在安全边际内持有/加仓 |
| 时间维度 | 中期（几周到几年） | 长期（永远，如果质量保持） |
| 杠杆态度 | 可以主动使用（已知break point时） | 避免（除极特殊情况） |
| 能力圈定义 | 宏观叙事 + 反射性判断 | 具体公司/行业商业模式 |
