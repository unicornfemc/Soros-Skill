# 信贷泡沫与杠杆动力学

## 信贷周期与反射性

信贷是现代金融体系中最强大的反射性放大器。

**核心机制：**
- 信贷扩张 → 资产价格上涨 → 抵押品价值增加 → 进一步信贷扩张
- 这个循环在去杠杆化时同样强力反转：
- 资产价格下跌 → 抵押品价值缩水 → 贷款方要求追加保证金 → 强制抛售 → 价格进一步下跌

---

## 泡沫的信贷阶段分类

| 阶段 | 特征 | 市场反应 |
|---|---|---|
| **初期** | 信贷从紧 → 宽松；企业/家庭开始借贷 | 温和上涨；基本面改善 |
| **膨胀期** | 信贷快速扩张；债务/GDP比率上升 | 强劲上涨；所有人盈利 |
| **狂热期** | 信贷标准下降；次级贷款/高杠杆产品涌现 | 极端上涨；杠杆使用普遍化 |
| **峰值** | 信贷增速触顶；新贷款开始减少 | 价格仍在高位；涨幅减缓 |
| **去杠杆** | 违约增加；贷款方收缩；强制清算 | 快速下跌；流动性枯竭 |

---

## 杠杆类型与风险

### 1. 场内杠杆（On-balance-sheet）
- 住房抵押贷款（低杠杆起步，高房价后转售可套现再融资）
- 企业杠杆收购（LBO）
- 特点：透明、可预测、强制清算线明确

### 2. 场外杠杆（Off-balance-sheet）
- 衍生品（CDS、CDO结构）
- 货币互换/远期合约
- 表外融资（SIVs、SPVs）
- 特点：透明度低；清算触发条件复杂；可能引发连锁反应

### 3. 影子银行杠杆
- 货币市场基金 → 商业票据
- 资产管理机构 → 杠杆ETFs
- 特点：看似低风险，实际流动性风险极高

| 指标 | 观察来源 | 预警阈值 |
|---|---|---|

| **影子银行规模** | FSB | 年增速 > 15% |

## 去杠杆化何时开始？

**三个触发条件：**

1. **外部冲击** → 资产价格下跌 → 抵押品价值缩水
2. **监管收紧** → 资本要求提升 → 银行收缩信贷
3. **信心丧失** → 贷款方拒绝展期 → 流动性枯竭

**去杠杆的类型：**

| 类型 | 机制 | 案例 |
|---|---|---|
| **通缩式去杠杆** | 债务人卖资产还债；资产价格下跌；债务比率不降反升 | 1930s美国、1990s日本 |
| **通胀式去杠杆** | 货币贬值；名义债务实际价值缩水；通胀抵消债务负担 | 二战后美国、2020s部分新兴市场 |
| **政策对冲去杠杆** | 央行大举购债；财政扩张；政府承担私人债务 | 2008后美欧英日 |

## 信贷泡沫中的索罗斯策略

**在泡沫形成期（识别偏见 + 确认反射循环）：**
- 做多被低估的资产类别
- 关注 BISES 测试中显示高反射性的领域

**在泡沫膨胀期（趋势确认 + 顺势加仓）：**
- 持有核心多头仓位
- 利用回调加仓

**在泡沫狂热期（寻找极点 + 准备做空）：**
- 逐步降低净多头
- 寻找反向交易机会
- 做空高相关性资产（如2007做空CDO权益层）

**在去杠杆初期（快速退出 + 流动性为王）：**
- 立即减持高风险资产
- 持有现金/主权债券
- 不要试图"接飞刀"

---

## 案例：2007-2008 全球金融危机

| 阶段 | 市场表现 | 索罗斯策略 |
|---|---|---|
| 2007初 | 次贷问题浮现 | 减少在金融股上的风险敞口 |
| 2007夏 | 两房问题暴露 | 做空CDO权益层 |
| 2008春 | Bear Stearns倒闭 | 持有黄金/避险资产 |
| 2008秋 | Lehman倒闭 | 持有现金；等待时机 |
| 2009春 | QE宣布 | 重仓买入；大幅获利 |

**关键洞察：** 泡沫顶峰时最危险的信号是"所有人都赚了很多钱"和"信贷标准彻底放松"。
