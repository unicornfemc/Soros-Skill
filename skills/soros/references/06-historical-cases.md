# 历史经典案例

## 1. 1992 — 打垮英格兰银行

### 背景
- 德国统一后为控通胀大幅加息（基准利率提至9%+）
- 英国经济衰退，无力跟随加息
- 英国在ERM中维持英镑/马克汇率 ≈ 2.95
- 德国马克实际升值，英镑实际高估约15%

### 索罗斯的分析框架
```
偏见：英国会不惜一切代价维持汇率稳定
反射循环：
  - 高利率压制英国经济 → 政治压力增加
  - 市场开始怀疑维持成本 → 英镑期货贴水
  - 央行越干预 → 市场越确信最终会放弃
Breaking point：英国央行弹药耗尽 + 政治意愿消失
```

### 交易
- 索罗斯：做空10亿英镑
- 其他投机者：闻风而动
- 1992年9月16日（黑色星期三）：英国退出ERM，英镑贬值15%
- **索罗斯单日盈利约10亿美元**

### 教训
> 当一个汇率有政治意愿维持却没有经济基础支撑时，最终经济基础会赢。

## 2. 1987 — 黑色星期一

### 背景
- 美股在1987年10月前已大幅上涨
- 程序化交易（Portfolio Insurance）大量存在
- 市场下跌触发更多卖单 → 踩踏

### 索罗斯的角色
- 在崩盘前持有大量多头
- 察觉到程序化止损的正反馈循环
- 在底部附近大幅买入
- 他后来承认："我在1987年赔了钱，但比大多数人少。"

### 反射性分析
- 1987年泡沫的反射性：
  - 股价上涨 → 程式交易规模扩大 → 股价继续上涨
  - 一旦下跌，程式卖单形成自我强化的下跌循环
- Portfolio Insurance 的设计者没有预见到自己的策略会加速崩盘

### 教训
> 你不需要知道泡沫何时破裂，但你需要知道你在泡沫中，并且有应对计划。

## 3. 1997-1998 — 亚洲金融危机

### 背景
- 泰国：固定汇率（泰铢/美元），大量短期外债
- 经常账户逆差，外汇储备不足
- 房地产泡沫，银行体系脆弱

### 索罗斯的分析
- 泰铢高估；固定汇率无法维持
- 反射性：
  - 外资持续流入 → 掩盖基本面脆弱性
  - 一旦外资停止流入，贬值预期自我实现
- Breaking point：外汇储备降至临界水平

### 交易
- 大量做空泰铢/泰股
- 危机从泰国蔓延至印尼、韩国、马来西亚
- **1997年成为"新兴市场教父"**

### 教训
- 固定汇率 + 开放资本账户 = 死亡组合
- 外债/外汇储备比率是核心风险指标
- 传染效应是亚洲危机的关键特征

## 4. 1999-2000 — 互联网泡沫

### 背景
- "新经济"叙事：互联网改变一切；旧经济估值模型失效
- NASDAQ从1998年10月到2000年3月上涨400%+
- 大量IPO没有盈利也可以定价数十亿美元

### 索罗斯的策略
- 在泡沫初期参与科技股
- 2000年初开始撤退
- 在泡沫顶峰做空部分科技股
- **成功规避了2000-2002年80%的跌幅**

### 反射性分析
- 互联网公司股价上涨 → 更多资金流入 → 更多互联网公司成立
- "注意力经济"逻辑自我强化
- 但没有现金流支撑的估值 → 不可持续
- Breaking point：盈利预期落空 + 利率上升

### 教训
> "泡沫在不破的时候是看不见的。只有破的时候，才知道那是泡沫。"

## 5. 2008 — 全球金融危机

### 背景
- 美联储长期低利率 → 房地产泡沫
- 金融创新（CDO、CDS）将风险分散/隐藏
- 影子银行体系杠杆极高

### 索罗斯的判断
- 早在2007年已识别到次贷泡沫
- 反射性：
  - 房价上涨 → 次贷发放标准下降 → 房价继续上涨
  - 金融机构的杠杆化掩盖了真实风险
- Breaking point：房价下跌 → 次贷违约 → CDO价值崩溃 → 雷曼倒闭

### 危机中的操作
- 2007年减少金融股敞口
- 2007-2008做空CDO相关资产
- 2009年3月美联储量化宽松后大量买入风险资产
- **在危机中保存资本，并在复苏时大幅获利**

### 教训
- "too big to fail" 问题
- 监管的滞后性
- 信贷泡沫的反射性比股市泡沫更难识别

## 历史案例的共性框架

| 案例 | 主导偏见 | 反射性机制 | Breaking Point |
|---|---|---|---|
| 1992英国 | 汇率稳定不可动摇 | 高利率压制经济 → 政治压力 | 外汇储备耗尽 |
| 1987美国 | 程序化交易控制风险 | 止损卖单 → 加速下跌 | 流动性枯竭 |
| 1997泰国 | 固定汇率可持续 | 外资流入掩盖脆弱性 | 储备降至临界 |
| 2000美股 | 互联网改变估值逻辑 | 资金流入 → 更多IPO | 盈利落空 + 加息 |
| 2008美国 | 房价只涨不跌 | 信贷扩张 → 杠杆累积 | 房价下跌 + 违约 |
