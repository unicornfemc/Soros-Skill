# 货币与固定收益套利

## 固定汇率机制与反射性

### 汇率制度的本质

**固定汇率是一种反射性承诺：**
- 政府宣布维持汇率在某个区间
- 市场开始相信这个承诺 → 汇率维持稳定
- 稳定本身成为"基本面"的一部分 → 投机者押注汇率不会偏离
- 但维持成本在不断累积（外汇储备消耗、利率倒挂）
- 最终，承诺无法兑现 → 汇率崩溃，幅度远超"正常"区间

**这就是索罗斯1992年英镑交易的逻辑。**

---

## ERM/ERM II 机制详解

**欧洲汇率机制（ERM）的工作原理：**
1. 成员国货币对德国马克有+/- 15%的波动区间
2. 各国央行有义务在外汇市场干预以维持区间
3. 当汇率接近区间边界，央行必须干预
4. 但央行弹药有限——外汇储备是有限的

**反射性陷阱：**
- 市场知道央行弹药有限
- 如果央行弹药耗尽，汇率将无法维持
- 投机者攻击的正是这个"弹药耗尽的预期"
- 央行越干预，市场越确信最终会失败
- 这种预期本身加速了央行弹药的消耗

## 货币攻击机制

### 第一类：基本面攻击（最经典）

**条件：**
- 汇率严重高估（经常账户持续逆差）
- 外汇储备不足（无法长期干预）
- 政策空间受限（利率已无调整空间）

**机制：**
1. 投机者借入被攻击货币
2. 在外汇市场大量抛售
3. 央行被迫动用外汇储备接盘
4. 外汇储备降至临界水平
5. 市场形成"央行即将投降"的预期
6. 抛售加速 → 央行放弃干预 → 汇率自由浮动
7. 贬值幅度远超合理水平（超调）

### 第二类：流动性攻击（影子杠杆）

**不需要基本面极端错位：**
1. 投机者通过衍生品建立大量空头头寸
2. 期权市场的隐含波动率飙升
3. 央行干预消耗外汇储备
4. 即使央行最终守住，市场也在短期内大幅波动
5. 投机者从期权波动率交易中获利

## 1992年英镑战役：完整复盘

### 背景
- 英国加入ERM，将英镑汇率与德国马克挂钩
- 德国统一后，为抑制通胀大幅加息
- 英国经济衰退，无法跟随德国加息
- 英镑汇率被严重高估

### 索罗斯的判断（1992年9月）
1. **偏见：** "英国不会让英镑贬值，因为信誉至关重要"
2. **反射性：** 维持汇率的成本（高利率压制经济）正在累积
3. **Breaking point：** 英国政府宁愿失去信誉也不愿让经济衰退加深
4. **结论：** 英镑必然贬值，只是时间问题

### 交易结构
- 借入10亿英镑（成本：约2.5%/年 = 2500万）
- 卖出英镑，买入德国马克
- 等待英国退出ERM
- 买回英镑归还，赚取汇率差价

### 结果
- 英镑贬值15%，索罗斯获利约10亿美元
- 成为"打垮英格兰银行的人"

## 固定收益套利

### 核心概念：收益率曲线与利差交易

**利差交易（Carry Trade）：**
- 借入低利率货币（融资货币）
- 买入高收益货币/资产
- 赚取利差收益

**反射性陷阱：**
- 利差交易受欢迎 → 资本流入高收益货币 → 本币升值 → 额外收益
- 这个正反馈循环吸引更多利差交易者
- 但一旦逆转，去杠杆化会导致踩踏：
- 卖出高收益资产 → 本币贬值 → 损失扩大 → 强制平仓 → 本币进一步贬值

### 利差交易逆转信号

| 信号 | 含义 |
|---|---|
| 央行意外加息 | 融资成本上升，利差收窄 |
| 全球风险偏好恶化 | 高收益货币被抛售 |
| 外汇储备下降 | 新兴市场央行无法维持汇率 |
| VIX飙升 | 风险平价策略去杠杆化 |

## 现代货币战与竞争性贬值

**当一国主动贬值时：**
1. 出口竞争力提升 → 贸易顺差增加（短期）
2. 但贬值预期形成 → 资本外流 → 本币进一步承压
3. 贸易对手国反击 → 竞争性贬值预期
4. 最终所有参与国都面临通胀压力

**索罗斯在这种情况下的分析框架：**
1. 哪个国家最先、最彻底地利用贬值？
2. 哪个国家最后被迫放弃汇率稳定？
3. 汇率战对黄金、大宗商品的影响？

## 汇率预测的关键指标

| 指标 | 数据来源 | 观察重点 |
|---|---|---|
| 外汇储备/月进口覆盖率 | 央行/BIS | < 3个月 = 危险区 |
| 经常账户/GDP | IMF/BIS | 持续逆差 = 脆弱性 |
| 外债/外汇储备 | 世界银行 | > 100% = 高风险 |
| 实际有效汇率（REER） | BIS/IMF | 极端高估/低估 |
| 央行政策利率 vs 核心国家 | 央行公告 | 利差扩大方向 |
| 资本流动数据 | IMF/BOP | 持续净流出 = 本币压力 |
